Linux

Pour jouer:
Cliquer Solo pour jouer contre l'ordinateur.
Cliquer Multijoueur pour jouer en ligne
-Le premier joueur clique Serveur.
-Le deuxieme joueur clique Client.
Cliquer sur Quitter pour quitter.
Comment jouer:
Utiliser les fleches pour jouer.
Vers le haut, monte la bar.
Vers le bas, descend la bar.
